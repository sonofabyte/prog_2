# Sticky Notes (App)

A small, lightweight note taking app.

## Usage

1. **Install Flask...**
1. type ```export FLASK_APP=app.py```, windows users use ```set FLASK_APP=app.py```
1. type ```flask run```

## Features

- Write plaintext notes with fancy titles
- Modify notes
- Delete notes
