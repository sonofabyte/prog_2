# Sticky Notes (App)

Eine kleine, einfache App für Notizen.

## Verwendung

1. **Install Flask...**
1. type ```export FLASK_APP=app.py```, windows users use ```set FLASK_APP=app.py```
1. type ```flask run```

## Funktionen

- Schreiben von Textnotizen mit  Titeln
- Ändern von Notizen
- Notizen löschen
